# FoxyProxy

## What It Does

**FoxyProxy** is a browser extension that helps users manage and switch between multiple proxy servers easily.
Instead of manually changing proxy settings in the browser, FoxyProxy allows you to quickly enable or disable proxies directly from the browser toolbar.

This is especially useful for cybersecurity learners who need to route browser traffic through security testing tools.

## Why It Is Useful

* Allows quick switching between **different proxy configurations**
* Helps route browser traffic through **security testing tools**
* Saves time by avoiding manual proxy configuration
* Essential for workflows involving **intercepting proxies**

## Key Features

* One-click **enable or disable proxy**
* Manage **multiple proxy profiles**
* Automatically match URLs to specific proxies
* Simple interface for **quick proxy switching**
* Compatible with tools used in security testing environments

## Installation

1. Open your browser extension store.
2. Search for **FoxyProxy**.
3. Click **Add to Browser** or **Install**.
4. Pin the extension to your browser toolbar.
5. Configure the proxy settings according to your testing setup.

## Use Case in Cybersecurity

Example scenario:

A cybersecurity learner wants to analyze web requests using **Burp Suite**.

Steps:

* Configure **Burp Suite** as a proxy server.
* Set the proxy in FoxyProxy.
* Enable the proxy using the extension.

Now all browser traffic passes through the proxy tool, allowing the learner to:

* Intercept HTTP requests
* Modify requests and responses
* Analyze application behavior
