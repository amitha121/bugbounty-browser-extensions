# IP Address and Domain Information

## What It Does

**IP Address and Domain Information** is a browser extension that provides detailed information about a website’s network and domain.

It helps users quickly view key details such as IP address, DNS records, hosting provider, and server location directly from the browser.

## Why It Is Useful

* Helps perform **basic reconnaissance on websites**
* Shows **IP address and hosting details**
* Displays **DNS and domain information**
* Useful for learning **OSINT and network analysis**
* Helps understand how websites are structured

## Key Features

* Displays **website IP address**
* Shows **server location and hosting provider**
* Provides **DNS records and domain details**
* Access to **WHOIS information**
* Quick overview of **network infrastructure**

## Installation

1. Open your browser extension store.
2. Search for **IP Address and Domain Information**.
3. Click **Add to Browser** or **Install**.
4. Pin the extension to your toolbar.
5. Visit any website and click the extension icon to view details.

## Use Case in Cybersecurity

Example scenario:

A learner visits a website and uses the extension to gather information.

The extension reveals:

* IP address
* Hosting provider
* DNS records
* Server location

Using this, the learner can:

* Perform **initial reconnaissance**
* Understand **target infrastructure**
* Prepare for further **security analysis**
